APIGuardian: Enterprise API Security GatewayAPIGuardian is a high-performance security layer designed to sit in front of your microservices. It handles robust API Key Generation, JWT Authentication, and dynamic Rate Limiting to protect your infrastructure from abuse, brute-force attacks, and unauthorized access.🚀 Key FeaturesSecure Key Management: Cryptographically secure API key generation with metadata support.Dynamic Rate Limiting: Implements the Token Bucket Algorithm to restrict request flow per user/key.JWT Integration: Full support for JSON Web Tokens for stateless user session security.Request Logging: Detailed audit trails of every incoming request, including latency and status codes.Database Persistence: Reliable storage using MySQL with optimized indexing for rapid key lookup.Glassmorphism Dashboard: A modern React-based UI for managing keys and monitoring traffic in real-time.🛠️ Tech StackBackend: Spring Boot, Spring Security, Hibernate (JPA).Frontend: React.js, Framer Motion (for animations), CSS Grid/Flexbox.Database: MySQL 8.0.Security: JJWT (Java JWT), BCrypt Password Encoding.🏗️ System ArchitectureAPIGuardian acts as a middleware between the client and your internal services. It intercepts every request to validate credentials and check rate-limit quotas before forwarding the traffic.📋 API Endpoints (Quick Reference)MethodEndpointDescriptionAuth RequiredPOST/api/auth/registerRegister a new developer accountNoPOST/api/keys/generateGenerate a new unique API KeyYes (JWT)GET/api/keys/my-keysFetch all keys belonging to the userYes (JWT)GET/api/secure/resourceA protected resource (Requires valid API Key)Yes (API Key)🚦 Getting StartedPrerequisitesJDK 17 or higherMaven 3.6+MySQL ServerInstallation & SetupClone the repositoryBashgit clone https://github.com/yourusername/api-guardian.git

🛡️ Rate Limiting LogicThe rate limiter uses a sophisticated algorithm to ensure fair usage. By default, the Pro Plan allows for 10,000 requests per day, while the Free Plan is limited to 100.
